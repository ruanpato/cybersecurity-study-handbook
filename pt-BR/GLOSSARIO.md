# Glossário

🌐 **Idioma:** **Português (pt-BR)** · [English (en-US)](../en-US/GLOSSARY.md) · [README](README.md)

Termos e siglas usados no handbook, em ordem alfabética. A primeira ocorrência de cada termo
no [HANDBOOK.md](HANDBOOK.md) também traz uma explicação no contexto.

- **ABAC** — *Attribute-Based Access Control*. Autorização baseada em atributos/contexto.
- **ACID** — Atomicidade, Consistência, Isolamento, Durabilidade (propriedades de transações).
- **ACL** — *Access Control List*. Lista de permissões associada a um objeto.
- **AD** — *Active Directory*. Serviço de diretório da Microsoft (domínios e florestas).
- **AppSec** — *Application Security*. Segurança de aplicações.
- **ARP** — *Address Resolution Protocol*. Mapeia IP para MAC na rede local.
- **ASLR** — *Address Space Layout Randomization*. Randomiza endereços de memória.
- **ASVS** — *Application Security Verification Standard* (OWASP).
- **ATT&CK** — base de conhecimento de táticas/técnicas de adversários (MITRE).
- **BEC** — *Business Email Compromise*. Fraude por comprometimento de e-mail corporativo.
- **BOLA** — *Broken Object Level Authorization*. Falha de autorização por objeto (APIs).
- **C2** — *Command and Control*. Canal de controle do atacante.
- **CAPEC** — *Common Attack Pattern Enumeration and Classification* (MITRE).
- **CIA** — Confidencialidade, Integridade, Disponibilidade.
- **CIS** — *Center for Internet Security*. Publica Controls e Benchmarks.
- **CNAPP** — *Cloud-Native Application Protection Platform*.
- **CORS** — *Cross-Origin Resource Sharing*. Relaxa a Same-Origin Policy de forma controlada.
- **CSP** — *Content Security Policy*. Cabeçalho que restringe origens de recursos.
- **CSPM** — *Cloud Security Posture Management*.
- **CSRF** — *Cross-Site Request Forgery*.
- **CTI** — *Cyber Threat Intelligence*.
- **CVE** — *Common Vulnerabilities and Exposures*. Identificador de vulnerabilidade.
- **CVSS** — *Common Vulnerability Scoring System*. Pontua severidade.
- **CWE** — *Common Weakness Enumeration*. Tipos de fraqueza.
- **CWPP** — *Cloud Workload Protection Platform*.
- **DAST** — *Dynamic Application Security Testing*.
- **DEP/NX** — *Data Execution Prevention* / *No-eXecute*. Impede execução em regiões de dados.
- **DFIR** — *Digital Forensics and Incident Response*.
- **DHCP** — *Dynamic Host Configuration Protocol*.
- **DMARC/DKIM/SPF** — mecanismos de autenticação de e-mail.
- **DNS** — *Domain Name System*. Resolução de nomes.
- **EDR/XDR** — *Endpoint/Extended Detection and Response*.
- **ELF** — *Executable and Linkable Format* (binários Linux).
- **GDPR** — *General Data Protection Regulation* (UE).
- **GPO** — *Group Policy Object* (Windows/AD).
- **GRC** — *Governance, Risk and Compliance*.
- **HMAC** — *Hash-based Message Authentication Code*.
- **HSM** — *Hardware Security Module*.
- **HSTS** — *HTTP Strict Transport Security*.
- **IAM** — *Identity and Access Management*.
- **IAST** — *Interactive Application Security Testing*.
- **IDOR** — *Insecure Direct Object Reference*.
- **IDS/IPS** — *Intrusion Detection/Prevention System*.
- **IOC** — *Indicator of Compromise*.
- **IaC** — *Infrastructure as Code*.
- **JIT** — *Just-in-Time* (acesso concedido só quando necessário).
- **JTAG** — interface de depuração/teste de hardware.
- **JWT** — *JSON Web Token*.
- **KDF** — *Key Derivation Function*.
- **KMS** — *Key Management Service*.
- **LGPD** — Lei Geral de Proteção de Dados (Brasil).
- **LotL** — *Living off the Land*. Abuso de ferramentas legítimas do sistema.
- **Mach-O** — formato de binário do macOS.
- **MFA** — *Multi-Factor Authentication*.
- **MISP** — plataforma open source de compartilhamento de inteligência de ameaças.
- **NAT** — *Network Address Translation*.
- **NTLM** — protocolo de autenticação legado da Microsoft.
- **OIDC** — *OpenID Connect*. Autenticação sobre OAuth 2.0.
- **OSINT** — *Open Source Intelligence*.
- **OWASP** — *Open Worldwide Application Security Project*.
- **PAM** — *Pluggable Authentication Modules* (Linux) ou *Privileged Access Management*
  (contexto de identidade). O significado depende do contexto.
- **PCI DSS** — *Payment Card Industry Data Security Standard*.
- **PE** — *Portable Executable* (binários Windows).
- **PKI** — *Public Key Infrastructure*.
- **PTES** — *Penetration Testing Execution Standard*.
- **RASP** — *Runtime Application Self-Protection*.
- **RBAC** — *Role-Based Access Control*.
- **ReDoS** — *Regular expression Denial of Service*.
- **RE** — *Reverse Engineering*.
- **REST** — estilo de arquitetura de APIs.
- **ROE** — *Rules of Engagement*. Regras de engajamento.
- **SAML** — *Security Assertion Markup Language*. Federação de identidade.
- **SAST** — *Static Application Security Testing*.
- **SBOM** — *Software Bill of Materials*.
- **SCA** — *Software Composition Analysis*.
- **SIEM** — *Security Information and Event Management*.
- **SIP** — *System Integrity Protection* (macOS).
- **SLSA** — *Supply-chain Levels for Software Artifacts*.
- **SOAR** — *Security Orchestration, Automation and Response*.
- **SOC** — *Security Operations Center*; também *SOC 2* (relatório de controles).
- **SOP** — *Same-Origin Policy*.
- **SQLi** — *SQL Injection*.
- **SSH** — *Secure Shell*.
- **SSO** — *Single Sign-On*.
- **SSRF** — *Server-Side Request Forgery*.
- **STIX/TAXII** — formato e transporte para compartilhar inteligência de ameaças.
- **STRIDE** — modelo de categorias de ameaça (Spoofing, Tampering, Repudiation, Information
  disclosure, Denial of service, Elevation of privilege).
- **SUID/SGID** — bits que executam um arquivo com privilégio do dono/grupo (Linux).
- **TCC** — *Transparency, Consent, and Control* (controle de acesso a recursos no macOS).
- **TCP/UDP** — protocolos de transporte (confiável / sem conexão).
- **TLS** — *Transport Layer Security*.
- **TOCTOU** — *Time-of-Check to Time-of-Use*. Condição de corrida.
- **TPM** — *Trusted Platform Module*.
- **TTP** — *Tactics, Techniques, and Procedures*.
- **UART** — interface serial usada em depuração de hardware.
- **WSTG** — *Web Security Testing Guide* (OWASP).
- **XSS** — *Cross-Site Scripting*.
- **XXE** — *XML External Entity*.
- **YARA** — linguagem de regras para identificar arquivos/memória por padrões.
- **Zero Trust** — modelo de segurança que nunca confia implicitamente; verifica sempre.

## Termos adicionais (v0.2)

- **blast radius** — o alcance do dano se um componente é comprometido; reduzido por menor
  privilégio e segmentação.
- **certificate pinning** — vincular um app a um certificado/chave específico do servidor para
  resistir a MITM.
- **corpus** — as entradas-semente que um fuzzer muta.
- **crash triage** — decidir se um crash é um bug real e relevante para segurança, e sua severidade.
- **deep link** — uma URL que abre uma tela/ação específica no app (valide no servidor).
- **DPIA / RIPD** — Relatório de Impacto à Proteção de Dados; documenta riscos de tratamento de
  dados pessoais de alto risco.
- **EPSS** — Exploit Prediction Scoring System; probabilidade estimada de uma vuln ser explorada.
- **firmware** — software armazenado num dispositivo que controla seu hardware.
- **fuzzing** — alimentar um alvo com entradas mutadas/aleatórias para achar crashes/bugs.
- **harness** — o pequeno programa que alimenta entradas na função sob teste de fuzz.
- **insecure deserialization** — reconstruir objetos a partir de dados não confiáveis de forma
  insegura.
- **JWKS** — JSON Web Key Set; chaves públicas do issuer usadas para verificar assinaturas de token.
- **KEV** — catálogo Known Exploited Vulnerabilities da CISA.
- **MASVS / MASTG** — OWASP Mobile Application Security Verification Standard / Testing Guide.
- **metamorfismo** — malware que reescreve o próprio código a cada geração.
- **NetworkPolicy** — recurso do Kubernetes que controla o tráfego entre pods.
- **PKCE** — Proof Key for Code Exchange; protege o fluxo authorization-code do OAuth.
- **polimorfismo** — malware que muda sua aparência (ex.: por packing) a cada infecção.
- **prepared statement** — query parametrizada que separa código SQL de dados (previne SQLi).
- **Purple Team** — red + blue colaborando para transformar ataques em detecções.
- **risco residual** — o risco que permanece após os controles.
- **matriz de risco** — tabela de probabilidade × impacto para priorizar riscos.
- **root cause analysis** — identificar a causa subjacente de um bug ou incidente.
- **simulação de engenharia social** — exercício autorizado para medir/melhorar a resiliência
  humana.
- **SWD** — Serial Wire Debug; interface de depuração de hardware (cf. JTAG).
- **timeline** — ordenação cronológica de eventos em DFIR.
- **gestão de vulnerabilidades** — o ciclo inventário → descoberta → priorização → remediação →
  verificação.
- **WebView** — componente de navegador embutido num app mobile/desktop.
