## English

This repository is, at once, an **introductory technical book**, a **phased study roadmap**, a
**catalog of hands-on labs and projects**, and a **map of specializations**. It is written to
be read on its own and also exported to PDF.

- Start here: **[en-US/README.md](en-US/README.md)**
- The book: **[en-US/HANDBOOK.md](en-US/HANDBOOK.md)**
- Study roadmap: **[en-US/ROADMAP.md](en-US/ROADMAP.md)**

**Ethics first.** All offensive content is educational and defensive, to be practiced only on
your own systems, isolated labs, or platforms that explicitly authorize testing. See
[en-US/HANDBOOK.md, Chapter 0](en-US/HANDBOOK.md#chapter-0).
