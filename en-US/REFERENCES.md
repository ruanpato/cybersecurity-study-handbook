## 1. Frameworks, standards, and catalogs (fundamental)

### NIST Cybersecurity Framework (CSF)

- **Author/Org.:** NIST (US) · **Type:** framework · **Level:** intro–intermediate ·
  **Subject:** governance and security functions.
- **Relevance:** a reference structure for security programs (see
  [HANDBOOK Ch. 25](HANDBOOK.md#chapter-25)).
- **Link:** https://www.nist.gov/cyberframework · **Version:** CSF 2.0 (current) · **Cost:** free ·
  **Priority:** fundamental · **Status:** Verified on 2026-06-18.

### NIST SP 800-53 — Security and Privacy Controls

- **Author/Org.:** NIST · **Type:** controls catalog · **Level:** intermediate ·
  **Subject:** security and privacy controls.
- **Link:** https://csrc.nist.gov/pubs/sp/800/53/r5/upd1/final · **Version:** Rev. 5 (final;
  released 2020, updates through 2020-12-10) · **Cost:** free ·
  **Priority:** recommended · **Status:** Verified on 2026-06-18.

### NIST SP 800-61 Rev. 3 — Incident Response Recommendations and Considerations

- **Author/Org.:** NIST · **Type:** guide · **Level:** intermediate · **Subject:** incident
  response ([Ch. 16](HANDBOOK.md#chapter-16)).
- **Link (Rev. 3, current):** https://csrc.nist.gov/pubs/sp/800/61/r3/final · **Version:** Rev. 3
  (final, April 2025) · **Cost:** free · **Priority:** fundamental ·
  **Status:** Verified on 2026-06-18 — **Rev. 2** was **withdrawn on 2025-04-03** and superseded
  by Rev. 3.
- **Historical note:** Rev. 2 (2012, "Computer Security Incident Handling Guide") still has
  conceptual value but is officially withdrawn: https://csrc.nist.gov/pubs/sp/800/61/r2/final
  (Verified on 2026-06-18).
